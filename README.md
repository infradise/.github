## üí° How It Works

This is a special repository named `.github`, provides default configurations for other repositories in this organization. Certain files placed here, such as `CONTRIBUTING.md` and `CODE_OF_CONDUCT.md`, will be automatically used by other repositories in this organization that do not have their own versions of these files. For more details, see the [official GitHub documentation](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file).
